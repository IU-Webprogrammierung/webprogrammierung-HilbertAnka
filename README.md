# 🚀 Projekt: Web-Programmierung - Portfolio einer Theatermalerin

## 📌 Projektbeschreibung
Diese Website dient als **berufliche und persönliche Online-Präsenz** für eine Theatermalerin.
Sie präsentiert den beruflichen Werdegang, vermittelt Einblicke in die Arbeitsweise und bietet eine Galerie mit ausgewählten Fotos aus dem Bereich der Theatermalerei.

## 🛠️ Technologien

- **HTML5** (semantische Tags)
- **CSS3** (Flexbox, CSS Grid, Media Queries)
- **Responsive Design**
- **Git & GitHub** (Versionskontrolle mit regelmäßigen Commits)
- **Barrierefreiheit**: Semantische HTML-Struktur, alt-Texte


## 📱 Layout & Responsivität

- **Breakpoint**: 768px (Mobile bis 767px, Desktop ab 768px)
- **Layouts**: Ein- und zweispaltig, abhängig von Bildschirmgröße
- **Verwendung von Flexbox und CSS Grid** für Anordnung

## 📄 Inhalte

- **Home**: Einführung, Galerie-Teaser, Video, Verlinkung zu weiteren Unterseiten
- **Gallery**: Fotos in Kacheln mit Links zu 6 weiteren Unterseiten
- **Studio Insights**: Schritt-für-Schritt-Erklärung der Prospektherstellung
- **About Me**: CV, Foto
- **Contact**: Kontaktformular
- **Impressum & Datenschutz**: Informationen zur Verantwortlichkeit und Datenverarbeitung
